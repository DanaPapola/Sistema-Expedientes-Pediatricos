# **Sistema de Expedientes Pediátricos**

## **Ingeniería de Requerimientos**

Proyecto desarrollado para el Departamento de Enfermería de la Universidad Autónoma de Aguascalientes (UAA), cuyo propósito es digitalizar la captura y administración de expedientes pediátricos utilizados durante las visitas domiciliarias realizadas por estudiantes de enfermería.

El sistema busca sustituir el proceso actual basado en formatos impresos, permitiendo almacenar la información en una base de datos centralizada, facilitar el seguimiento de pacientes y generar reportes para apoyar la toma de decisiones.

---

# **Objetivo**

Desarrollar un prototipo web que permita:

* Capturar valoraciones pediátricas.  
* Registrar el consentimiento informado.  
* Administrar expedientes.  
* Calcular automáticamente un semáforo de riesgo.  
* Generar reportes para profesores.  
* Gestionar distintos tipos de usuarios.

# **Navegación del prototipo**

El prototipo puede recorrerse desde la pantalla de inicio.

Flujo principal:

Inicio de sesión  
        ↓  
Panel principal  
        ↓  
Consentimiento informado  
        ↓  
Captura de valoración  
        ↓  
Consulta de expedientes  
        ↓  
Semáforo de riesgo  
        ↓  
Reportes

También existen vistas específicas para:

* Alumno  
* Profesor

---

# **Funcionalidades principales**

* Inicio de sesión mediante usuario y contraseña.  
* Captura de valoración pediátrica.  
* Consentimiento informado digital.  
* Consulta de expedientes.  
* Semáforo automático de riesgo.  
* Reportes consolidados.  
* Gestión de usuarios.

Estas funcionalidades corresponden a los requerimientos funcionales definidos y validados durante el proyecto.
